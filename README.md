## Getting Started

Para conseguir executar o projeto, clone o projeto e siga os passos abaixo.

### Instalação

1. You must have the following tools:

- Node.js + NPM
- React Native
- JDK 8
- Android Studio / Genymotion para emular

2. Run Commands: 

```sh
npm install
```

```sh
npm start --reset-cache
```
**Android**

```sh
npm run android 
```

**IOS**

```sh
npm run ios
```
